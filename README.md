# Кати-Ряд

Match-3 игра с уникальной механикой: вместо того, чтобы двигать фишки по одной,
игрок наклоняет всё поле в одну из четырёх сторон. Фишки катятся под "гравитацией",
сталкиваются и складываются в ряды по 3+ — те исчезают, остальные снова катятся.
Получаются комбо-цепочки. Цель — выжить как можно дольше.

## Как играть

- **Десктоп:** стрелки `←↑→↓` или клавиши `W A S D`. Также можно тащить мышью.
- **Телефон:** свайп пальцем по полю в нужную сторону.
- **Любая платформа:** кнопки-стрелки по сторонам поля.

3 и более одинаковых фишек в ряд (по горизонтали или вертикали) исчезают и приносят очки.
Чем длиннее комбо-цепочка в одном ходу, тем больше множитель очков (×1, ×2, ×3...).
Каждый ход на поле добавляются 3 новые фишки. Когда поле забивается и наклон в любую
сторону уже не двигает фишки — игра окончена. Рекорд сохраняется локально.

## Запуск локально

Игра использует ES-модули, поэтому открыть `index.html` напрямую с `file://` не получится.
Подними простой статический сервер из папки проекта:

```bash
python3 -m http.server 8000
```

И открой `http://localhost:8000`.

Альтернатива: расширение Live Server для VS Code.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub и запушь содержимое этой папки в ветку `main`.
2. В настройках репозитория: **Settings → Pages → Source → Deploy from a branch**, выбери `main` / `(root)`.
3. Через 1–2 минуты игра будет доступна по адресу `https://<username>.github.io/<repo>/`.

## Тесты

Логика игры покрыта unit-тестами на чистом `node:test`:

```bash
node --test tests/
```

## Структура

- `index.html` — разметка
- `style.css` — стили и анимации
- `logic.js` — чистая игровая логика (тестируется)
- `game.js` — состояние, рендер, ввод
- `tests/logic.test.js` — тесты
